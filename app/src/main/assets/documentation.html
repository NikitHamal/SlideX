SlideX Documentation

Introduction

Welcome to the SlideX documentation. Here you will find information about how to use the app, including details about the JSON structure for creating slides.

JSON Structure

The slides are generated from a JSON object. The basic structure of the JSON is as follows: 


{
  "elements": [
    {
      "type": "text",
      "text": "Hello, World!",
      "x": 50,
      "y": 50,
      "size": 24,
      "color": "#000000"
    },
    {
      "type": "image",
      "url": "https://example.com/image.png",
      "x": 100,
      "y": 100,
      "width": 100,
      "height": 100
    }
  ]
}

Elements

The elements array contains a list of objects, each representing an element on the slide. Each element has a type property, which can be one of the following:

  • text
  • image
  • shape
  • table
  • chart
  • icon

Text Element

The text element has the following properties:

  • text: The text to display.
  • x: The x-coordinate of the text.
  • y: The y-coordinate of the text.
  • size: The font size of the text.
  • color: The color of the text in hex format.

Image Element

The image element has the following properties:

  • url: The URL of the image.
  • x: The x-coordinate of the image.
  • y: The y-coordinate of the image.
  • width: The width of the image.
  • height: The height of the image.
