SlidesAI Documentation

Introduction

Welcome to the SlidesAI documentation. This guide provides detailed information on how to use the app, with a focus on the JSON structure for creating slides and leveraging the AI capabilities.

AI Chat

The Chat tab allows you to generate slides using natural language prompts. You can select from different AI models to generate the slide content.

  • Gemini: Google's powerful AI model.
  • Qwen: A new experimental AI model.

To generate a slide, simply type a prompt in the chat box, such as "Create a slide about the solar system", and the AI will generate the corresponding JSON for you.

JSON Structure

All slides are defined by a JSON object. This allows for precise control over every element on the slide.

Root Object

The root JSON object has the following properties:

  • backgroundColor: (Optional) A hex color string (e.g., "#FFFFFF") for the slide's background.
  • elements: An array of element objects that will be rendered on the slide.

Elements

The elements array contains a list of objects, each representing an element on the slide. Each element must have a type property. 


{
  "backgroundColor": "#F0F4F8",
  "elements": [
    {
      "type": "text",
      "text": "Welcome to SlidesAI",
      "x": 20, "y": 20, "width": 280, "height": 40,
      "fontSize": 28, "color": "#34495e", "fontWeight": "bold"
    },
    {
      "type": "image",
      "url": "https://i.imgur.com/6J2z1J2.png",
      "x": 40, "y": 80, "width": 240, "height": 100,
      "cornerRadius": 10
    }
  ]
}

Element Types

Text Element

  • type: "text"
  • text: The string content to display.
  • x, y, width, height: Position and dimensions.
  • fontSize: The font size in sp.
  • color: Hex color string.
  • fontWeight: (Optional) "bold" or "normal".
  • italic: (Optional) true or false.
  • alignment: (Optional) "left", "center", or "right".

Image Element

  • type: "image"
  • url: The URL of the image.
  • x, y, width, height: Position and dimensions.
  • cornerRadius: (Optional) The radius for rounded corners.

Shape Element

  • type: "shape"
  • shapeType: "rectangle", "oval", "line", "triangle", "star", or "hexagon".
  • x, y, width, height: Position and dimensions.
  • color: Hex color string for the fill.
  • cornerRadius: (Optional) For rectangles.
  • opacity: (Optional) From 0.0 to 1.0.
  • strokeWidth, strokeColor: (Optional) For the element's border.

Icon Element

  • type: "icon"
  • iconName: The name of the Material Icon (e.g., "settings", "home").
  • x, y, width, height: Position and dimensions.
  • color: Hex color string.

Chart Element

  • type: "chart"
  • chartType: "bar", "pie", or "line".
  • data: An array of data points, each with label (string) and value (number).
  • showLegend: (Optional) true or false.

Table Element

  • type: "table"
  • x, y, width, height: Position and dimensions.
  • rows, columns: Number of rows and columns.
  • data: A 2D array of strings for the cell content.
  • headerColor, cellColor, borderColor: (Optional) Hex color strings.
  • borderWidth: (Optional) The width of the border.

AI Agent Instructions

When building or modifying the app, please adhere to the following guidelines:

  • The slide rendering is handled by SlideRenderer.java. All element drawing logic is contained within the respective element classes (e.g., TextElement.java).
  • The Gemini AI is integrated via NetworkManager.java.
  • The Qwen AI is integrated via QwenManager.java in the com.slides.ai.qwen package.
  • The UI is defined in XML layouts and managed by the corresponding Activity or Fragment classes.
  • Before submitting, ensure that all new features are documented in this file.
