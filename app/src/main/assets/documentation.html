Welcome to SlidesAI! This guide will help you create stunning presentations with ease. You can create slides manually using our JSON editor or let our AI do the heavy lifting for you.

ü§ñ Using the AI Chat

The Chat tab is your creative partner. Simply tell the AI what you want, and it will generate a slide for you.

Examples of prompts:

  • "Create a title slide for a presentation on marketing trends."
  • "Make a slide with a bar chart showing sales data for the last quarter."
  • "Generate a slide with a timeline of major events in World War II."

Tip: Be as specific as you can in your prompts for the best results. Mention colors, layouts, and the type of content you want to see.

üìù The JSON Editor

For those who want full control, the Code tab provides a powerful JSON editor. Every slide is a JSON object, giving you the power to customize every detail.

Root Object

The main JSON object has two key properties:

  • backgroundColor: (Optional) Sets the background color of your slide. Use a hex code like "#FFFFFF" for white.
  • elements: This is an array where you define all the elements that will appear on your slide.

Example Slide


{
  "backgroundColor": "#FDFEFE",
  "elements": [
    {
      "type": "text",
      "text": "The Future of AI",
      "x": 20, "y": 20, "width": 280, "height": 40,
      "fontSize": 32, "color": "#2C3E50", "fontWeight": "bold"
    },
    {
      "type": "image",
      "url": "https://images.unsplash.com/photo-1593349480503-685d3a5c54c4",
      "x": 40, "y": 80, "width": 240, "height": 120,
      "cornerRadius": 15
    }
  ]
}

üé® Element Types

Here are the different types of elements you can add to your slides:

Text

For titles, bullet points, and paragraphs.

  • type: "text"
  • text: The text you want to display.
  • x, y, width, height: The position and size of the text box.
  • fontSize: The size of the text.
  • color: The color of the text (hex code).
  • fontWeight: (Optional) Set to "bold" for bold text.
  • italic: (Optional) Set to true for italic text.
  • alignment: (Optional) "left", "center", or "right".

Image

Add visuals to your slides.

  • type: "image"
  • url: The URL of the image.
  • x, y, width, height: The position and size of the image.
  • cornerRadius: (Optional) Creates rounded corners for the image.

Shape

Use shapes to add design elements or highlight information.

  • type: "shape"
  • shapeType: "rectangle", "oval", "line", "triangle", "star", or "hexagon".
  • x, y, width, height: The position and size of the shape.
  • color: The fill color of the shape.
  • cornerRadius: (Optional) For rectangles.
  • opacity: (Optional) A value from 0.0 (transparent) to 1.0 (opaque).
  • strokeWidth, strokeColor: (Optional) For the shape's border.

Icon

Add Material Icons to your slides.

  • type: "icon"
  • iconName: The name of the icon (e.g., "settings", "home", "star").
  • x, y, width, height: The position and size of the icon.
  • color: The color of the icon.

Tip: You can find a full list of Material Icon names at the official Google Fonts website.

‚ú® Tips & Tricks

  • Layering: Elements are drawn in the order they appear in the elements array. The last element in the array will be on top.
  • Experiment: Don't be afraid to experiment with different element combinations and properties to create unique and engaging slides.
  • AI as a Starter: Use the AI to generate a base slide, then fine-tune the JSON in the Code editor to get it just right.
